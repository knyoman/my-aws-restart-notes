## 🚀 Tentang Repositori Ini

Repositori ini adalah jurnal teknis (*technical journal*) saya selama mengikuti program intensif **AWS re/Start**. Di sini, saya mendokumentasikan setiap teori, perintah *Command Line*, skrip Python, serta arsitektur AWS yang saya bangun di lingkungan *hands-on lab*. 

Tujuan utama program ini adalah mempersiapkan fondasi yang kuat untuk peran seperti *Cloud Practitioner, Cloud Engineer*, atau *DevOps Junior*, selaras dengan standar sertifikasi **AWS Certified Cloud Practitioner (CLF-C02)**.

## 📚 Kurikulum & Hands-on Labs

*Klik pada masing-masing modul di bawah ini untuk melihat detail materi dan lab yang telah saya selesaikan:*

<details>
<summary><b>1. Cloud Foundations 🌐</b></summary>
<br>
  
  * Konsep komputasi awan, infrastruktur global AWS, dan AWS Pricing.
  * *Shared Responsibility Model*.
  * **Lab:** Pengenalan Amazon S3 & Amazon EC2.
</details>

<details>
<summary><b>2. Linux Administration 🐧</b></summary>
<br>
  
  * CLI navigation, Users & Groups management, File Permissions.
  * Process Management, Services monitoring, dan Log Files analysis.
  * **Lab:** Pembuatan *Bash Shell Scripting* untuk otomatisasi tugas server.
</details>

<details>
<summary><b>3. Networking & Security 🛡️</b></summary>
<br>
  
  * IP Subnetting, OSI Layer, dan protokol jaringan.
  * Keamanan infrastruktur (Network & System Hardening).
  * **Lab:** Konfigurasi **Amazon VPC** (Public/Private Subnets) dan implementasi kebijakan **AWS IAM**.
</details>

<details>
<summary><b>4. Python Programming & DevOps 🐍</b></summary>
<br>
  
  * Sintaks dasar, *Flow Control*, *Functions*, dan *Modules*.
  * Pemrograman untuk administrasi sistem (*System Admin with Python*).
  * Pengenalan metodologi CI/CD dan otomatisasi.
</details>

<details>
<summary><b>5. Databases 🗄️</b></summary>
<br>
  
  * Relational vs Non-Relational Databases.
  * Perintah SQL (DDL & DML).
  * **Lab:** Implementasi dan interaksi dengan **Amazon RDS**, Aurora, dan **Amazon DynamoDB**.
</details>

<details>
<summary><b>6. AWS Architecture, Scaling & Serverless 🚀</b></summary>
<br>
  
  * AWS Well-Architected Framework.
  * High Availability menggunakan **Auto Scaling** dan **Elastic Load Balancing (ELB)**.
  * Resolusi nama dengan **Amazon Route 53**.
  * **Lab:** Membangun arsitektur tanpa server (*Serverless*) menggunakan **AWS Lambda** dan API Gateway.
</details>

<details>
<summary><b>7. Infrastructure as Code (IaC) & Monitoring 🤖</b></summary>
<br>
  
  * Monitoring sistem secara *real-time* menggunakan **Amazon CloudWatch** dan CloudTrail.
  * Menggunakan JSON/YAML untuk otomatisasi infrastruktur.
  * **Lab:** Melakukan *deploy* arsitektur berulang (*Repeatable Deployments*) menggunakan **AWS CloudFormation**.
</details>

<details>
<summary><b>8. Generative AI & Machine Learning 🧠</b></summary>
<br>
  
  * Fondasi Machine Learning dan prinsip *Responsible AI*.
  * *Prompt Engineering* dasar.
  * **Lab:** Eksplorasi **Amazon Bedrock**, melatih model dengan **Amazon SageMaker**, dan menggunakan AI *No-code* via **PartyRock**.
</details>
